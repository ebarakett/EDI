3D Car Reflection

This is a reflection my process making a 3D model of a toy car. Enjoy!

Guidelines:

  • The car's design will be comprised of the intersection of 3 different forms
  • The car will be designed in a 1:1 ration 2
    • The creator will aim for size accuracy within 1% of the car's actual size
  • The car's design file will be exported as STL
  • The model will be uploaded and oriented in Cura for printing
    • Support material should be minimized
  • The file should be saved as a gcode
  • The final design is well documented, and the design process is reflected on

Process:

  • Creating the Engineering Drawing
    • The car was drawn to scale using calipers
    • After the drawing was finalized, it was scanned
    • Screenshots were taken for each side of the car drawn
  • 3D Modeling the car
    • Each face of the car was composed on every plane side
    • Negative space was removed using the extrusion tool
    • Detailing was added using the extrusion and fillet tools
    • The file was exported as STL file
  • Printing the model
    • The design was uploaded in Cura
    • Final adjustments were made and support material was minimized
    • The nozzle was set to 0.6mm
    • The file was exported as a GCODE
    • The model was printed
  • Documenting the process
    • A CAD drawing was made off of the 3D model and used to calculate the margin of error
    • The design was imported into Modelo
    • This reflection was created as an HTML document

Setbacks and Solutions:

  • One setback was scaling the drawing incorrectly
    • Solution: the 3D model was resized to fit the dimensions of the smallest scaled face
  • Another setback was doing the project from home and not having the example models to get inspiration from
    • Solution: I had to depend fully on the photos I had taken of the toy car prior to leaving
